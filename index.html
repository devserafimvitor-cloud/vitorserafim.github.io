🚀 Projetos

✈️ Sistema de Controle de Voo

Aplicação desenvolvida para simular o controle de aproximação de aeronaves, analisando variáveis críticas como altitude e velocidade. O sistema aplica validações e regras para orientar decisões seguras de pouso, incluindo alertas em tempo real para ajustes operacionais.

📊 Sistema de Avaliação de Alunos

Sistema completo de avaliação acadêmica com entrada validada de dados, cálculo de médias e aplicação de regras hierárquicas para classificação do desempenho do aluno. Implementa cenários como aprovação, recuperação e reprovação, incluindo análise de exceções e condições especiais.

🧾 Sistema de Classificação por Idade

Sistema desenvolvido para classificação de usuários com base na faixa etária, aplicando regras condicionais para determinar permissões e restrições de acesso. Focado em lógica estruturada e organização de decisões.
